Blog 

Funcionalidades:

Crear autores

Crear categorías

Crear posts

Buscar posts por título

Plantillas HTML con una base común

Formularios generados con Django Forms

Panel de administración de Django


1_Entrar al proyecto

cd mi_blog


2_Crear entorno virtual (opcional)

python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\Scripts\activate      # Windows


3_Instalar dependencias

pip install django


4_Ejecutar el servidor

python manage.py runserver


Abrir en el navegador:

http://127.0.0.1:8000/
